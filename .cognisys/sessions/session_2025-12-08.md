## Cloud Integration Plan (Approved)

### Architecture Decision
Transform CogniSys from single-drop-directory to distributed multi-source library:
- Local sources (Downloads, Documents, Desktop)
- Network sources (NAS, SMB shares)
- Cloud sources (OneDrive, Google Drive, iCloud, Proton Drive)
- Both mounted folder AND native API support
- Two-way sync capability

### Implementation Phases
1. Phase 1 (Days 1-2): Mounted folder detection
2. Phase 2 (Days 3-5): FileSource abstraction layer
3. Phase 3 (Days 6-10): OneDrive native API
4. Phase 4 (Days 11+): Additional providers

### New Modules Planned
- cognisys/cloud/ - Cloud detection, sync, OAuth
- cognisys/storage/ - FileSource abstraction layer
